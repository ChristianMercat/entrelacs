3. Aufgaben

Alle Fassungen dieses Artikels: [Deutsch] [English] [Español] [français] [עברית] [italiano] [Português do Brasil]

Drucken Sie diese Seite aus

Exercices 1

und versuchen Sie sich daran.

Wenden Sie die vier vorher beschriebenen Schritte konsequent an:

  1. Setzen Sie ein Kreuz in die Mitte jeder Seite.
  2. Verbinden Sie die Enden miteinander.
  3. Entscheiden Sie, welche Fäden oben, welche unten liegen.
  4. Verbreitern Sie die Linie.

Denken Sie daran: jedes Kreuz befindet sich in der Mitte einer Seite, und jede Seite enthält ein Kreuz.

Eine Seite = ein Kreuz

Und um die Enden zu verbinden, wiederholen Sie:

Folgen Sie der Wand, gehen Sie um die Ecke, folgen Sie der Wand.

Diese Zeichnungen in K3D-Format für die Software von Steven Abbott.

Wenn es Ihnen nicht auf Anhieb gelingen sollte, verzweifeln Sie nicht. Probieren Sie es noch einmal und schauen Sie dann in den Lösungen nach.
