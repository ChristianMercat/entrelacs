e tente sua mão sobre eles. Aplique as quatro etapas anteriormente descritas:

 # Coloque uma travessia no meio de cada aresta.
 # Conecte os pedaços uns com os outros usando o labirinto.
 # Enfatize a característica abaixo / acima com o guia.
 # Amplie a estrada, primeiro as pontes, em seguida, cada lado da estrada.

file in K3D formato per Steven Abbott software.

Lembre-se, cada travessia ocorre no meio de uma aresta e cada aresta tem um cruzamento.

Uma aresta = um cruzamento

E para conectar os pedaços, repita:

Siga a parede, vire a esquina, siga a parede.

Se você não tiver sucesso na primeira vez, não se desespere e tenha uma olhada na resposta.
